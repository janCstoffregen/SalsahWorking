Welcome to Salsah — a web-based virtual research environment that helps researchers in the humanities to deal with digital sources. It allows the annotation and linkage of arbitrary sources such as digital texts, still and moving images, and sound.

SALSAH is one (generic) user interface to handle the data from the main repository called Knora.
